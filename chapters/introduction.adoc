== 简介

代码分析过程分为词法分析和语法分析两个部分:

* 词法分析, 把文本输入转换成 Token 流.
* 语法分析, 把 Token 流转换成 AST.

理解 Elixir 编译过程， 词法分析和语法分析是绕不过的.

=== 词法分析

Leex 生成的词法分析器, 读取输入, 匹配输入与指定的正则表达式, 执行匹配所关联的 Erlang 代码.

词法分析是在输入中寻找字符的模式, 正则表达式是对模式描述的工具.

词法分析器内部是一个 DFA.
